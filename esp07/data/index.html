Description

This project uses an ESP8266 module to send commands to a CH9329 USB keyboard emulator, enabling it to simulate keystrokes.

Functionality

  • Configure the reverse shell by specifying the target IP address and port.
  • Send keyboard input using CH9329 HID commands.
